How to config xdebug

File -> Settings->Languages & Frameworks -> PHP -> Debug

Install browser toolbar

Enable listening for PHP Debug Conections

File -> Settings->Languages & Frameworks -> PHP -> Servers:

Configure absolute path on the Servers

Your computer (host) absolute path on the Servers (inside the container)
/path/to/current/project/ /var/www/html/
/path/to/current/project/pub/ /var/www/html/pub/

Para poder usar XDEBUG por consola en necesario configurar PHP_IDE_CONFIG con el mismo nombre que usamos en ServerName

Example:

PHP_IDE_CONFIG=serverName=magento2.com

How to use Composer

you can execute composer inside the php container by ssh console runing the command warp php ssh

You can execute composer without entering the container, run: warp composer

for more info please run: warp composer --help
